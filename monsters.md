# 游戏怪物种类文档

## 概述

本游戏包含多种不同类型的怪物，每种怪物具有独特的属性和解锁条件。怪物的属性包括生命值、攻击力、移动速度等，部分怪物还具备远程攻击能力。

## 怪物解锁规则

怪物根据**侵蚀等级**逐步解锁：

| 侵蚀等级 | 解锁怪物 |
|----------|----------|
| 0 | 裂变蠕虫 |
| 1 | 畸变射手 |
| 2 | 甲壳巨兽、幽能织网者 |
| 3 | 熵变暴君 |
| 4 | 暗影突袭者 |
| 5 | 能量护盾者 |
| 6 | 虚空领主 |

## 游戏限制

- **最大怪物类型数**：全部（同时活跃，除去boss）
- **最大怪物总数**：100只
- **最大远程怪物数**：10只
